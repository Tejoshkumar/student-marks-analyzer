# Student Marks Analyzer

A Python-based console application that analyzes student marks using
lists and nested dictionaries.

## Description
This project allows users to enter details of multiple students along
with their subject-wise marks. It then performs analysis such as
maximum marks, minimum marks, total marks, average marks, and identifies
the class topper based on total marks.

## Features
- Supports multiple students
- Supports multiple subjects per student
- Student-wise analysis:
  - Maximum marks
  - Minimum marks
  - Total marks
  - Average marks
- Class-level analysis:
  - Class topper based on total marks

## Concepts Used
- Python lists
- Nested dictionaries
- Loops
- Conditional statements
- Built-in functions (max, min, sum)
